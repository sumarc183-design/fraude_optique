# Note de confidentialité

Ce dossier est destiné à un GitHub personnel ou à un portfolio public.

Il doit rester limité à une description anonymisée des travaux. Ne pas y ajouter :

- jeux de données ;
- modèles `.pkl`, `.joblib`, MLflow ou autres artefacts entraînés ;
- archives d'artefacts MLflow `.zip` ;
- rapports internes complets ;
- métriques détaillées issues de données réelles ;
- noms d'entreprise, de client, de projet, de dataset ou d'endpoint ;
- captures d'écran d'outils internes ;
- logs, payloads API ou exports de production ;
- fichiers `.env`, tokens, clés d'API ou identifiants ;
- code métier contenant des règles internes.

Avant publication, relire le README et vérifier qu'il ne permet pas d'identifier une organisation, un système interne, un volume métier précis ou une donnée sensible.
